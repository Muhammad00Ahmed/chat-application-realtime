# Real-time Chat Application

Real-time chat application with WebSocket connections, typing indicators, message history, and user presence.

## Features

- Real-time messaging
- Typing indicators
- User presence (online/offline)
- Message history
- Private messaging
- Group chats
- File sharing
- Emoji support
- Read receipts

## Tech Stack

- Node.js
- Express.js
- Socket.io
- MongoDB
- React.js

## Installation

```bash
npm install
npm run dev
```
